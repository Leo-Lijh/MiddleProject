中期项目：消消乐

模块说明：

    -controller: 控制器包，存放业务逻辑处理模块
        -operDao.py: 用户操作处理模块(李佳和、高嘉阳)
        -scoreDao.py: 分数处理模块(陈建桥)
        -userDao.py: 用户相关模块(张贝勒)
    -model: 存放实体类
        -user.py: 用户实体类(陈建桥)
        -vector.py: 坐标点实体类
    -page: 存放页面视图(html)
        
    -util: 存放基本业务模块
        -config.py: 配置文件模块
        -DBUtil.py: 数据库操作模块(刘涛)
        -httpServer.py: http服务模块(范珊珊)
        -sql.py: sql语句模块
    -main.py: 程序运行主入口
    
1、配置文件和sql语句根据自己需要自行添加

2、模块中方法根据需要自行添加，但必须添加详细注释

3、模块头部注释中的@editor为该模块编写者名字
